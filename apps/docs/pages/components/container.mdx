# Container

A layout component that centers content and constrains its maximum width. Perfect for creating consistent page layouts with proper responsive padding.

## Import

```js
import { Container } from '@beauginbey/vanilla-components';
```

## Basic Usage

The `Container` component centers its content and provides responsive padding.

<LiveExample
  code={`import { Container, Box, Text } from '@beauginbey/vanilla-components';

export default function App() {
  return (
    <Box backgroundColor={2} p={4}>
      <Container>
        <Box p={4} backgroundColor={1} borderRadius="lg">
          <Text size="xl" weight="bold" mb={2}>
            Centered Content
          </Text>
          <Text color="secondary">
            This content is wrapped in a Container component, which centers it 
            and constrains its maximum width. The container also provides 
            responsive padding on the sides.
          </Text>
        </Box>
      </Container>
    </Box>
  );
}`}
/>

## Size Variants

Containers come in different max-width sizes: `sm`, `md`, `lg`, and `xl`.

<LiveExample
  code={`import { Container, Box, Text, Flex } from '@beauginbey/vanilla-components';

export default function App() {
  const sizes = [
    { size: 'sm', maxWidth: '640px' },
    { size: 'md', maxWidth: '768px' },
    { size: 'lg', maxWidth: '1024px' },
    { size: 'xl', maxWidth: '1280px' },
  ];

  return (
    <Flex flexDirection="column" gap={4}>
      {sizes.map(({ size, maxWidth }) => (
        <Box key={size} backgroundColor={2} p={4}>
          <Container size={size}>
            <Box p={4} backgroundColor={1} borderRadius="lg">
              <Text weight="semibold">
                {size} Container ({maxWidth})
              </Text>
            </Box>
          </Container>
        </Box>
      ))}
    </Flex>
  );
}`}
/>

## Fluid Container

Use the `fluid` prop to remove the max-width constraint while keeping the padding.

<LiveExample
  code={`import { Container, Box, Text } from '@beauginbey/vanilla-components';

export default function App() {
  return (
    <Box>
      <Box backgroundColor={2} p={4} mb={4}>
        <Container>
          <Box p={4} backgroundColor={1} borderRadius="lg">
            <Text weight="semibold">Normal Container</Text>
            <Text size="sm" color="secondary">
              Has a maximum width constraint
            </Text>
          </Box>
        </Container>
      </Box>

      <Box backgroundColor={3} p={4}>
        <Container fluid>
          <Box p={4} backgroundColor={1} borderRadius="lg">
            <Text weight="semibold">Fluid Container</Text>
            <Text size="sm" color="secondary">
              No maximum width - expands to fill available space
            </Text>
          </Box>
        </Container>
      </Box>
    </Box>
  );
}`}
/>

## Nested Containers

Containers can be nested for complex layouts.

<LiveExample
  code={`import { Container, Box, Text, Flex } from '@beauginbey/vanilla-components';

  return (
    <Box backgroundColor={2} py={6}>
      <Container size="xl">
        <Text as="h1" size="3xl" weight="bold" mb={6} textAlign="center">
          Dashboard
        </Text>
        
        <Flex gap={4} flexDirection={{ mobile: 'column', tablet: 'row' }}>
          <Box flex={1}>
            <Container size="sm" fluid>
              <Box p={4} backgroundColor={1} borderRadius="lg">
                <Text weight="semibold" mb={2}>Sidebar</Text>
                <Text size="sm" color="secondary">
                  Nested container with fluid width
                </Text>
              </Box>
            </Container>
          </Box>
          
          <Box flex={2}>
            <Container size="md" fluid>
              <Box p={4} backgroundColor={1} borderRadius="lg">
                <Text weight="semibold" mb={2}>Main Content</Text>
                <Text size="sm" color="secondary">
                  Another nested container for the main content area
                </Text>
              </Box>
            </Container>
          </Box>
        </Flex>
      </Container>
    </Box>
  );
}`}
/>

## Props

`Container` accepts all `Box` props plus the following:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'lg'` | Maximum width size |
| `fluid` | `boolean` | `false` | Remove max-width constraint |

The default responsive padding adjusts based on screen size for optimal reading experience.
