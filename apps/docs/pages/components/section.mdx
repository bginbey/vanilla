# Section

A semantic layout component for organizing page content into distinct sections with consistent spacing.

## Import

```js
import { Section } from '@beauginbey/vanilla-components';
```

## Basic Usage

The `Section` component provides semantic HTML5 `<section>` elements with built-in spacing.

<LiveExample
  code={`import { Section, Container, Text } from '@beauginbey/vanilla-components';

export default function App() {
  return (
    <div>
      <Section>
        <Container>
          <Text as="h2" size="2xl" weight="bold" mb={3}>
            About Us
          </Text>
          <Text color="secondary">
            This is a section with default padding. Sections help organize 
            content into meaningful groups while maintaining consistent spacing.
          </Text>
        </Container>
      </Section>

      <Section backgroundColor={2}>
        <Container>
          <Text as="h2" size="2xl" weight="bold" mb={3}>
            Our Services
          </Text>
          <Text color="secondary">
            Sections can have different background colors to create visual separation.
          </Text>
        </Container>
      </Section>
    </div>
  );
}`}
/>

## Size Variants

Sections come with different padding sizes: `sm`, `md`, `lg`, and `xl`.

<LiveExample
  code={`import { Section, Container, Text, Box } from '@beauginbey/vanilla-components';

export default function App() {
  const sizes = ['sm', 'md', 'lg', 'xl'];

  return (
    <Box>
      {sizes.map((size, index) => (
        <Section 
          key={size} 
          size={size}
          backgroundColor={index % 2 === 0 ? 1 : 2}
        >
          <Container>
            <Text weight="semibold">
              {size} Section
            </Text>
            <Text size="sm" color="secondary">
              This section has {size} padding
            </Text>
          </Container>
        </Section>
      ))}
    </Box>
  );
}`}
/>

## Alternating Backgrounds

Create visual rhythm with alternating section backgrounds.

<LiveExample
  code={`import { Section, Container, Text, Box } from '@beauginbey/vanilla-components';

export default function App() {
  const sections = [
    { title: 'Introduction', content: 'Start with a brief overview of your topic.' },
    { title: 'Main Points', content: 'Discuss the key points in detail.' },
    { title: 'Examples', content: 'Provide concrete examples to illustrate your points.' },
    { title: 'Conclusion', content: 'Summarize and provide next steps.' },
  ];

  return (
    <Box>
      {sections.map((section, index) => (
        <Section 
          key={section.title}
          backgroundColor={index % 2 === 0 ? 1 : 2}
        >
          <Container size="md">
            <Text as="h2" size="2xl" weight="bold" mb={3}>
              {section.title}
            </Text>
            <Text color="secondary">
              {section.content}
            </Text>
          </Container>
        </Section>
      ))}
    </Box>
  );
}`}
/>

          </Text>

/>

## Props

`Section` accepts all `Box` props plus the following:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Predefined padding size |
| `as` | `React.ElementType` | `'section'` | HTML element to render |

The component renders as a semantic HTML5 `<section>` element by default, which helps with document structure and accessibility.
