# Components

A comprehensive collection of accessible, customizable React components built with TypeScript and vanilla-extract.

## Component Library

### Layout
- [Box](/components/box) - The foundation of our layout system with access to all design tokens
- [Text](/components/text) - Polymorphic text component for consistent typography

### Buttons & Actions
- [Button](/components/button) - Flexible button with variants, sizes, and icon support
- [IconButton](/components/icon-button) - Square button designed specifically for icons

### Forms
- [Checkbox](/components/checkbox) - Styled checkbox with indeterminate state support
- [FormField](/components/form-field) - Wrapper for form controls with labels and error messages
- [Input](/components/input) - Text input with multiple variants and states
- [Radio](/components/radio) - Styled radio button for single selections
- [Select](/components/select) - Dropdown select with consistent styling
- [Switch](/components/switch) - Toggle switch for binary choices

### Icons & Media
- [Icon](/components/icon) - Flexible icon component with built-in gallery

## Getting Started

```bash
npm install @beauginbey/vanilla-components
```

```tsx
import { Button, Box, Text } from '@beauginbey/vanilla-components';

function App() {
  return (
    <Box p={4}>
      <Text as="h1" size="3xl" weight="bold" mb={4}>
        Welcome to Vanilla
      </Text>
      <Button variant="solid">
        Get Started
      </Button>
    </Box>
  );
}
```

## Design Principles

1. **Composability** - Components are designed to work together seamlessly
2. **Flexibility** - Extensive customization through props and variants
3. **Performance** - Zero runtime CSS-in-JS with vanilla-extract
4. **Developer Experience** - Intuitive APIs with excellent TypeScript support
5. **Accessibility** - WCAG compliant with proper semantic HTML
