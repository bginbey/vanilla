# Components

Vanilla Design System provides a comprehensive set of flexible, accessible components built with React and TypeScript.

## Layout Components

### Box

The Box component is the foundation of our layout system. It provides access to design tokens through the styling props.

<LiveExample
  code={`import { Box, Text } from '@beauginbey/vanilla-components';

export default function App() {
  return (
    <Box 
      p={4} 
      backgroundColor="secondary" 
      borderRadius="lg"
      display="flex"
      flexDirection="column"
      gap={3}
    >
      <Box p={3} backgroundColor="primary" borderRadius="md">
        <Text>I'm inside a Box!</Text>
      </Box>
      <Box p={3} backgroundColor="tertiary" borderRadius="md">
        <Text>Boxes can be nested and styled</Text>
      </Box>
    </Box>
  );
}`}
/>

#### Box Props

- `as` - Render as a different HTML element (default: `div`)
- `p`, `pt`, `pb`, `pl`, `pr`, `px`, `py` - Padding utilities
- `m`, `mt`, `mb`, `ml`, `mr`, `mx`, `my` - Margin utilities
- `display` - CSS display property
- `flexDirection`, `justifyContent`, `alignItems`, `gap` - Flexbox utilities
- `backgroundColor`, `color`, `borderColor` - Color utilities
- `borderRadius`, `boxShadow` - Visual utilities
- `style` - Additional CSS styles
- `minHeight`, `minWidth`, `maxHeight`, `maxWidth`, `height`, `width` - Dimension utilities

## Typography

### Text

A polymorphic text component for rendering typography with consistent styles.

<LiveExample
  code={`import { Text, Box } from '@beauginbey/vanilla-components';

  return (
    <Box display="flex" flexDirection="column" gap={3}>

      </Text>
      
      <Text size="lg" color="secondary">
        Large secondary text for subtitles
      </Text>
      
      <Text>
        Default body text with <Text as="span" weight="semibold">inline emphasis</Text> and
        <Text as="span" color="brand"> brand color</Text>.
      </Text>
      
      <Box backgroundColor="primary" p={3} borderRadius="md">
        <Text color="inverse">
          Inverse text on dark background
        </Text>
      </Box>
      
      <Box display="flex" gap={2}>
        <Text color="success">Success</Text>
        <Text color="warning">Warning</Text>
        <Text color="error">Error</Text>
        <Text color="info">Info</Text>

/>

#### Text Props

- `as` - HTML element to render (default: `span`)
- `size` - Text size from `xs` to `5xl`
- `weight` - Font weight: `normal`, `medium`, `semibold`, or `bold`
- `color` - Text color from theme
- `align` - Text alignment: `left`, `center`, or `right`
- `truncate` - Truncate text with ellipsis

### FormField

A wrapper component for form inputs with labels, helper text, and error messages.

<LiveExample
  code={`import { FormField, Input, Select, Box } from '@beauginbey/vanilla-components';
import { useState } from 'react';

export default function App() {
  const [email, setEmail] = useState('');
  const [hasError, setHasError] = useState(false);
  
  return (
    <Box display="flex" flexDirection="column" gap={4}>
      <FormField
        label="Email Address"
        helper="We'll never share your email"
        required
      >
        <Input 
          type="email" 
          placeholder="you@example.com"
          value={email}
          onChange={(e) => {
            setEmail(e.target.value);
            setHasError(!e.target.value.includes('@'));
          }}
          error={hasError}
        />
      </FormField>
      
      <FormField
        label="Country"
        error={hasError ? "Please enter a valid email first" : undefined}
      >
        <Select disabled={hasError}>
          <option value="">Select a country</option>
          <option value="us">United States</option>
          <option value="uk">United Kingdom</option>
          <option value="ca">Canada</option>
        </Select>
      </FormField>
      
      <FormField
        label="Password"
        helper="Must be at least 8 characters"
        required
      >
        <Input type="password" placeholder="Enter password" />
      </FormField>
    </Box>
  );
}`}
/>

#### FormField Props

- `label` - Field label text
- `helper` - Helper text shown below the input
- `error` - Error message (replaces helper text when present)
- `required` - Show required indicator
- `children` - The form control (Input, Select, etc.)

## Accessibility

All components in the Vanilla Design System are built with accessibility in mind:

- **Keyboard Navigation**: All interactive components support keyboard navigation
- **ARIA Labels**: Proper ARIA attributes are included where needed
- **Focus Management**: Clear focus indicators that respect `prefers-reduced-motion`
- **Semantic HTML**: Components render appropriate HTML elements
- **Screen Reader Support**: Components work well with assistive technologies

### Icon Props

- `icon` - The icon component to render (required)
- `size` - Icon size: `xs`, `sm`, `md`, `lg`, `xl`, or custom value

- `stroke` - Stroke width (if supported by icon library)
- `label` - Accessible label for the icon
- `className` - Additional CSS classes
- `style` - Inline styles

## TypeScript Support

All components are fully typed with TypeScript, providing excellent IDE support and type safety:

```typescript
import { Button, ButtonProps } from '@beauginbey/vanilla-components';

// Type-safe props
const MyButton = (props: ButtonProps<'button'>) => {
  return <Button {...props} />;
};

// Polymorphic typing
const LinkButton = () => {
  return (
    <Button as="a" href="/path">
      This is type-safe as an anchor
    </Button>
  );
};
```
